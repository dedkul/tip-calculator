# Tip Calculator — iOS App

A simple iOS tip calculator app built with Swift, SwiftUI, and Xcode. Enter a bill, pick a tip, and split it between friends. That's it!

---

## How This Was Built (A VibeCode AI Experiment)

This was a small personal project to try out **AI-assisted app development** — sometimes called "vibe coding."

Here's the process I used:

1. **Asked Claude (AI) to write a full requirements document** for the app idea — covering screens, features, data, and step-by-step build instructions.
2. **Created a new project in Xcode**, then opened that same project folder in **Cursor** — an AI-powered code editor built on top of VS Code.
3. **Set up Cursor for iOS development** — installed the relevant Swift/iOS plugins inside Cursor, and used **Homebrew** to install the extra tools needed to make everything work smoothly.
4. **Followed the build steps from the requirements doc one by one** inside Cursor. Each step referenced a specific feature (like `B-003 → F-001`), so I always knew exactly what to ask Cursor's AI to build next.
5. Kept going step by step until the app was done — with no major errors along the way.

The whole point was to see how far you could get by just describing what you want, rather than writing code from scratch yourself. Turns out — pretty far!

## Tech Stack

- **Swift** — the coding language
- **SwiftUI** — for building the screens
- **Xcode** — where the app was built and tested

---

## Features

- Enter a bill amount
- Choose a tip (15%, 18%, 20%, or custom %)
- See the tip and total instantly
- Split the bill between multiple people
- Dark mode support
- Reset with one tap

---

## Requirements

- iOS 16 or later
- No internet needed — works fully offline
